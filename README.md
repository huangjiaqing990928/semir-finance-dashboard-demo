# 森马集团数字化财务驾驶舱 Demo

这是一个用于面试展示的“数字化财务”示样作品，定位为财务部门自用的差旅费用运营驾驶舱。

## 展示重点

- 用 3600 条模拟报销/出差数据展示数据清洗后的 BI 看板能力。
- 首页先说明“它能帮财务部做什么”：承接报销数据、看清费用分布、形成复核动作。
- 通过中国地图热力图呈现各地差旅热点和城市单据情况，并叠加城市光点、差旅动线和城市热度 Top 5。
- 支持月份、部门、费用类型、风险等级筛选，并联动 KPI、图表、AI 洞察和明细表。
- AI 财务洞察为规则型 Demo，后续可接企业知识库、大模型接口、报销系统、发票影像和 ERP。

## 演示话术建议

1. 先从顶部说明切入：这是给财务中心看的费用运营工具，不是 IT 后台系统。
2. 再讲中间三步：从报销系统或 Excel 接入数据，统一清洗成标准字段，再用 BI 看板看热点城市、部门费用和异常单据。
3. 地图区域重点讲“颜色越亮代表费用越集中，发光城市代表单据密度高，动线代表主要出差去向”，让吴经理一眼看到全国差旅态势。
4. 最后演示筛选条件：切换部门或风险等级后，KPI、地图、图表、AI 洞察和明细表会同步变化，说明后续可以接真实报销系统。

## 后续可扩展方向

- 接入真实费控/报销系统接口，替换当前模拟数据。
- 增加 Excel/CSV 导入，把财务部临时表快速变成看板。
- 接企业知识库和大模型接口，生成异常说明、复核建议、月结差异分析。
- 扩展权限、审批流、月结进度看板、供应商/门店/区域费用专题分析。
